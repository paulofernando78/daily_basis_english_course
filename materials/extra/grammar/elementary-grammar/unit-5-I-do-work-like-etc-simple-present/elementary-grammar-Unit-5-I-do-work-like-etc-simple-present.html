true verdadeiro(a) adjective

truth(s) verdade(s) noun substabtivo


We use the simple present for:


General truths


I like big cities.

Your English is very good. You speak very well.

People need food.


Scientific facts

The sun sets in the west.

The earth goes around the sun. A terra vira em torno sol.

Water boils at 100 degree celsius. A água ferve a 100 graus Celsius.

Stars explode when they die forming a supernova.


Routines


I wake up early every day.

On week days I wake up 6:30 to go to school.

Tom starts work at 7:30 and finishes at 8:00 at night.

Tom works very hard. Tom trabalha muito.

We do a lot of different things in our free time.


Não se omite o pronome "it".


It rains a lot in London. Chove muito em Londres.

It costs a lot of money to build a hospital. Custa muito caro construir um hospital.

This computer = It works very well. Este computador funciona muito bem.

The story = It goes like this. A história é assim.

rule regra


Rule #1. -s:


call -> calls x

eat -> eats x

look -> looks x

read -> reads x

run -> runs x

sleep -> sleeps x

talk -> talks x

walk -> walks x

work -> works x


come -> comes x

drive -> drives x

like -> likes x

live -> lives x

love -> loves x

make -> makes x

move -> moves x

practice -> practices x

save -> saves x

see -> sees x

use -> uses x

write -> writes x


Rule #2. If a verb ends with -o, -ch, -sh, -s, -x, and -z, add -es:


-o

do -> does x

go -> goes x


-ch

itch -> itches x

march -> marches x

teach -> teaches x

watch -> watches x


-sh

finish -> finishes x

wash -> washes x


-s

guess -> guesses adivinhar

kiss -> kisses x

miss -> misses x

pass -> passes x


-x

fix -> fixes x

mix -> mixes x

buzz -> buzzes x


Rule #3. When -y is preceded by a consonant, replace it with -ies:


carry -> carries x • x

cry -> cries x

fly -> flies x

hurry -> hurries x

marry -> marries x

party -> parties dar uma festa • They party all the time. Eles festejam o tempo todo.

reply -> replies x • x

study -> studies x • x

try -> tries x • x


But when the verb is preceded by a vowel, add -s:


enjoy -> enjoys • Nick enjoys video games. Nick gosta de video games.

lay -> lays • The hen lays eggs. A galinha bota ovos.

pay -> pays • John pays the bill. O john paga a conta.

play -> plays • Mike plays soccer on weekend. O Mike joga bola no fim de semana.

say /seɪ/ -> says /sez/ s • "I'm happy". Kate says "Estou feliz". A Kate diz.

stay -> stays • Mary stays at school after classes end. A Mary fica/permanece na escola depois que as aulas terminam.


Irregular

have -> has ter • Joe has a big house.
